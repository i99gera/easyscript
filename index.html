easyscript - make code faster and easier!


dependences:

    python3
    gcc

How to use:

tested successfully on linux only.
on windows failed tests
exec this comand to run your code in your code's folder:

python3 /path/to/easyscript file-with-your-code

How to code:


first programm:


print('hello world')

it will print 'hello world' string.

about variables:

supported main types:

int, str, char, float, double

supported prefixes:

l: long, c: const, u: unsigned

types are created using prefix and main type, for example:

ullint q,w,e

will declare vars q,w,e of type long long unsigned int.
order of prefixes doesn't matrer.

print and scan:


these functions are for input/output.
example:

int q,w
scan(q,w)
q+=w
print(q)

this code will scan two numbers and print sum of them.
note: print and scan can work with variables and functions only.
You can't print(q+1).

if

condition:
if condition is integer, it is understood as testing if condition in not 0
syntax example:

int q scan(q) if q%3==0: print('divided by 3') elif q%3==1: print('divided by 3 with remainder 1') else: print('divided by 3 with remainder 2')

this code finds remainder of division by 3
another example:

int q scan(q) if q%7: print(q,' is not divided by 7') else: print(q,' is divided by 7')

this code tests if number is divided by 7
