download easyscript


easyscript - make code faster and easier!


dependences:

    python3

How to run code:

python3 /path/to/easyscript file-with-your-code

If run failed you can compile to c++ code using:

python3 /path/to/easyscript -compile file-with-your-code

Learning by example:

you can copy any example and run it

calculating the sum of two numbers
scan(int q,int w) #scanning two integer numbers

q+=w #calculating sum of them

print(q) #printing the sum
sum of number from 1 to n
scan(int number) #reading n

sum=0 #here will be the sum

for iter<number: #it will iterate from 0 to number-1 

 sum+=iter+1 #iter is from 0 to number-1 but we need from 1 to number

print(sum) #write out the sum
another algirithm:
scan(int number) #reading n

sum=0 #here will be the sum

for 1=iter<number+1: #it will iterate from 1 to number 

 sum+=iter

print(sum)
and normal method:
scan(int number) #reading n

number=(number+1)*number/2 #it's the fastest method

print(sum)
