download easyscript


easyscript - make code faster and easier!




dependences:

    python3
    gcc


How to use:

tested on linux only.
exec this comand to run your code in your code's folder:
    python3 /path/to/easyscript file-with-your-code

How to code:

first programm:
    print('hello world')
  it will print 'hello world' string.
about variables:
  supported main types:
    int, str, char, float, double
  supported prefixes:
    l: long, c: const, u: unsigned
  types are created using prefix and main type, for example:
    ullint q,w,e
  will declare vars q,w,e of type long long unsigned int.
  order of prefixes doesn't matrer.
print and scan:
  these functions are for input/output.
  example:
    int q,w
    scan(q,w)
    q+=w
    print(q)
  this code will scan two numbers and print sum of them.
note: print and scan can work with variables and functions only. You can't print(q+1).
